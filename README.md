# Red Button Studios - Digital Portfolio Card

A blazing-fast, single-page digital portfolio built with pure HTML, CSS, and JavaScript. No frameworks. Just results.

## Features

- **Performance First**: Optimized for sub-2-second load times
- **Pure Vanilla Code**: HTML5, CSS3, JavaScript (ES6+)
- **Responsive Design**: Mobile-first approach with fluid typography
- **Accessibility**: WCAG 2.1 AA compliant with reduced motion support
- **3D Glass Orb**: Mesmerizing animated orb with cursor interaction
- **Glassmorphism UI**: Subtle, elegant glass card effects
- **Smooth Animations**: GPU-accelerated, 60fps animations

## Quick Start

1. Open `index.html` in a browser, or
2. Serve with any static server:
   ```bash
   npx serve .
   ```

## Customization

### Colors
Edit CSS custom properties in `:root` section of `style.css`:
```css
--color-magenta: #FF006E;
--color-cyan: #00F5FF;
--color-purple: #9D4EDD;
```

### Content
- Update text content in `index.html`
- Replace placeholder project screenshots in `/images/projects/`
- Update contact links (email, LinkedIn, GitHub)
